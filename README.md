# 📖 BLOG_APP <a name="about-project"></a>

** Blog App ** is a classic example of a blog website. A fully functional website that show the list of posts and empower readers to interact with them by adding comments and liking posts...

- Add and list Posts
- Add comments on posts
- User will be able to like a post

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- A text editor preferably [Visual Studio code](https://code.visualstudio.com/)
- Latest version of [Ruby](https://www.ruby-lang.org/en/downloads/) installed
- [PostgreSQL Server](https://www.postgresql.org/download/)


### Setup

Clone this [repository](https://github.com/Fayob/blog_app) to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:Fayob/blog_app.git
  cd blog_app
```

### Install

Install this project with:

```sh
  gem install
```
